## Learning Path

This example demonstrates how to:

1. **Structure a KMP application** with clean architecture
2. **Implement authentication** across client and server
3. **Share business logic** between platforms
4. **Create responsive UI** with Compose Multiplatform
5. **Build REST APIs** with Ktor
6. **Manage databases** with SQLDelight
7. **Configure dependency injection** with Koin
